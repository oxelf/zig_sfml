# SFML ported to the zig build system

This allows for seamless cross compilation.
(atleast from mac and linux to windows for now)

An example project using this build system can be found at:
https://github.com/oxelf/zig-sfml-template

## Dependencies
- SFML [c97c2ef](https://github.com/SFML/SFML)
- flac  [9547dbc](https://github.com/xiph/flac) build.zig in /flac
- vorbis/ogg [851cce9](https://github.com/xiph/vorbis) build.zig in /vorbis
- sheenbidi [1c65509](https://github.com/Tehreer/SheenBidi) build.zig in /sheenbidi
- freetype [9d847af](https://github.com/allyourcodebase/freetype)
- harfbuzz [6bb522d22cee0ce1d1446bd59fa5b0417f25303e](https://github.com/allyourcodebase/harfbuzz/)
